# Consulta de Ciudades por País

## Descripción

Esta aplicación web permite seleccionar un país y visualizar las ciudades disponibles en la base de datos junto con su población.

El sistema utiliza la base de datos **world** y muestra:

- Top 10 ciudades más pobladas del país seleccionado
- Top 10 ciudades con menor población

## Tecnologías utilizadas

- PHP 8
- MySQL
- HTML
- CSS

## Funcionamiento

1. El usuario selecciona un país del menú desplegable.
2. El sistema consulta la base de datos.
3. Se muestran las ciudades del país con su población.

## Base de datos

Se utiliza la base de datos **world**, con las tablas:

- country
- city
- countrylanguage
